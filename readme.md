### Installing

1. Clone this repository

    ```bash
    git clone https://github.com/JohnyKovalenko1337/uwatch_test.git
    ```
2.  Use the package manager [npm](http://www.npmjs.com/) to install dependencies.

    ```bash
    npm install
    ```
